## Question Types

### Multiple Choice Questions (MCQ)

Support for single and multiple correct answers with flexible configuration:

```json MCQ Configuration
{
  "type": "multiple_choice",
  "question": "Which of the following are React hooks?",
  "options": [
    "useState",
    "componentDidMount",
    "useEffect",
    "render"
  ],
  "correctAnswers": [0, 2],
  "multipleCorrect": true,
  "marks": 5,
  "explanation": "useState and useEffect are React hooks introduced in React 16.8"
}
```

**Features:**
- Single or multiple correct answers
- Option randomization
- Rich text formatting
- Image and media support
- Custom explanations

### Short Answer Questions

Text-based responses with automatic and manual grading options:

```json Short Answer Question
{
  "type": "short_answer",
  "question": "Explain the difference between let and var in JavaScript",
  "maxLength": 500,
  "keywords": ["scope", "hoisting", "block", "function"],
  "marks": 10,
  "gradingType": "manual"
}
```

### Essay Questions

Long-form responses with rubric-based grading:

```json Essay Question
{
  "type": "essay",
  "question": "Discuss the advantages and disadvantages of microservices architecture",
  "minWords": 300,
  "maxWords": 1000,

    "analysis": { "weight": 40, "description": "Provides thoughtful analysis" },
```

### Advanced Configuration

#### Question Randomization

Control how questions are presented to ensure fairness:

```json Randomization Settings
{
  "randomizeQuestions": true,
  "randomizeOptions": true,
  "questionPools": {
    "javascript_basics": {
      "questions": 20,
      "selectRandom": 5
    },
    "react_advanced": {
      "questions": 15,
      "selectRandom": 3
    }
  }
}
```

#### Adaptive Testing

Dynamic question selection based on candidate performance:

```json Adaptive Configuration
{
  "adaptiveTesting": {
    "enabled": true,
    "initialDifficulty": "medium",
    "adjustmentFactor": 0.5,
    "confidenceThreshold": 0.95,
    "maxQuestions": 30,
    "minQuestions": 15
  }
}
```

## Integration Capabilities

### API Integration

Programmatically create and manage assessments:

```javascript Assessment API Usage
// Create assessment
const assessment = await skillment.assessments.create({
  title: "React Developer Assessment",
  duration: 90,
  questions: [
    {
      type: "multiple_choice",
      question: "What is JSX?",
      options: ["JavaScript XML", "Java Syntax Extension", "JSON Extended"],
      correctAnswer: 0
    }
  ],
  settings: {
    randomizeQuestions: true,
    proctoring: true
  }
});

// Add questions from question bank
await skillment.assessments.addQuestions(assessment.id, {
  pool: "react_advanced",
  count: 5,
  difficulty: "medium"
});
```

### Security Configuration

1. **Risk Assessment**: Evaluate the importance and sensitivity of your assessment
2. **Proportional Security**: Apply appropriate security measures without hindering user experience
3. **Clear Communication**: Inform candidates about security requirements beforehand
4. **Testing**: Test all security features in candidate environments before deployment

### Performance Optimization

- **Question Loading**: Use lazy loading for large assessments
- **Media Optimization**: Optimize images and videos for fast loading
- **Caching**: Implement appropriate caching strategies
- **Monitoring**: Monitor assessment performance and loading times
