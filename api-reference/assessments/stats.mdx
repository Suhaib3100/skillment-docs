## Overview

Retrieve comprehensive statistics and analytics for an assessment.

## Endpoint

```
GET /api/assessments/{assessmentId}/stats
```

## Headers

```
Authorization: Bearer <access_token>
```

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `assessmentId` | string | Unique identifier of the assessment |

## Query Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `period` | string | Time period for stats (day, week, month, year) | `month` |
| `startDate` | string | Start date (ISO format) | - |
| `endDate` | string | End date (ISO format) | - |

## Status Codes

- `200` - Statistics retrieved successfully
- `401` - Unauthorized
- `404` - Assessment not found
- `500` - Internal server error
