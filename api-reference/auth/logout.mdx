## Overview

Logout user and invalidate current access tokens.

## Endpoint

```
POST /api/auth/logout
```

## Headers

```
Authorization: Bearer <access_token>
```

## Response

```json
{
  "success": true,
  "message": "Logout successful"
}
```

## Status Codes

- `200` - Logout successful
- `401` - Unauthorized
- `500` - Internal server error
