## Quick Start Guide

Get up and running with Skillment API in minutes:

<Steps>
  <Step title="Authentication">
    Register your organization and get your API credentials
    ```bash
    curl -X POST https://api.skillment.in/api/auth/register \
      -H "Content-Type: application/json" \
      -d '{"email": "admin@company.com", "orgName": "my-company", ...}'
    ```
  </Step>
  <Step title="Create Assessment">
    Build your first assessment with questions and test cases
    ```bash
    curl -X POST https://api.skillment.in/api/assessments \
      -H "Authorization: Bearer YOUR_TOKEN" \
      -H "Content-Type: application/json" \
      -d '{"title": "JavaScript Developer Test", "type": "coding", ...}'
    ```
  </Step>
  <Step title="Add Participants">
    Invite candidates to take your assessment
    ```bash
    curl -X POST https://api.skillment.in/api/participants \
      -H "Authorization: Bearer YOUR_TOKEN" \
      -d '{"firstName": "John", "lastName": "Doe", "email": "john@company.com"}'
    ```
  </Step>
</Steps>

## Architecture Overview

Skillment is built as a modern, scalable platform:

- **Backend API**: RESTful API built with Express.js and Prisma ORM
- **Real-time Code Execution**: WebSocket-based code runner supporting multiple languages
- **Multi-tenant**: Organization-based isolation with secure data separation
- **Scalable**: Designed to handle thousands of concurrent assessments
