## Getting Started

<AccordionGroup>
  <Accordion title="What is Skillment?">
    Skillment is a comprehensive assessment and event management platform designed for organizations that need to conduct secure, scalable, and insightful evaluations. It combines advanced proctoring, multiple question types, team collaboration, and detailed analytics in one platform.
  </Accordion>

  <Accordion title="How do I get started with Skillment?">
    Getting started is simple:
    1. Sign up for a Skillment account
    2. Create your organization and customize branding
    3. Build your first assessment with our question builder
    4. Invite candidates and configure security settings
    5. Monitor results with real-time analytics
  </Accordion>

  <Accordion title="What plans are available?">
    Skillment offers two main plans:
    - **Free Plan**: Basic features for small teams (up to 50 candidates/month)
    - **Elite Plan**: Advanced features including proctoring, unlimited candidates, and priority support
    - **Enterprise**: Custom solutions with dedicated support and advanced integrations
  </Accordion>

  <Accordion title="Can I try Skillment before purchasing?">
    Yes! We offer a 14-day free trial of the Elite plan with all features included. No credit card required to start your trial.
  </Accordion>
</AccordionGroup>

## Assessments

<AccordionGroup>
  <Accordion title="What types of questions can I create?">
    Skillment supports various question types:
    - **Multiple Choice**: Single or multiple correct answers
    - **Short Answer**: Text-based responses with keyword matching
    - **Essay Questions**: Long-form responses with rubric grading
    - **Coding Questions**: Programming challenges with automated testing
    - **File Upload**: Portfolio submissions and project reviews
  </Accordion>

  <Accordion title="Can I randomize questions and options?">
    Yes! Skillment offers comprehensive randomization options:
    - Randomize question order for each candidate
    - Shuffle multiple-choice options
    - Create question pools and randomly select questions
    - Adaptive testing based on candidate performance
  </Accordion>

  <Accordion title="How do I prevent cheating?">
    Skillment provides multiple security layers:
    - **Basic**: Full-screen mode, tab switch detection, time limits
    - **Standard**: Webcam monitoring, screen recording, face detection
    - **Advanced**: ID verification, room scan, AI-powered behavior analysis
    - **Browser Security**: Disable copy-paste, right-click, and developer tools
  </Accordion>

  <Accordion title="Can I create assessments collaboratively?">
    Absolutely! Team features include:
    - Multiple collaborators on assessments
    - Role-based permissions (admin, member, viewer)
    - Review and approval workflows
    - Version control and change tracking
  </Accordion>
</AccordionGroup>

## Proctoring & Security

<AccordionGroup>
  <Accordion title="How does proctoring work?">
    Skillment's proctoring system uses AI-powered monitoring:
    1. **Pre-assessment**: ID verification and environment check
    2. **During assessment**: Real-time webcam and screen monitoring
    3. **AI Analysis**: Automatic detection of suspicious behaviors
    4. **Alerts**: Real-time notifications for proctors
    5. **Post-assessment**: Detailed violation reports and recordings
  </Accordion>

  <Accordion title="What browsers support proctoring?">
    Proctoring works on:
    - **Chrome** (recommended) - Full feature support
    - **Firefox** - Full feature support
    - **Safari** - Most features supported
    - **Edge** - Full feature support
    
    Mobile browsers have limited proctoring capabilities.
  </Accordion>

  <Accordion title="Is candidate data secure and private?">
    Yes, security and privacy are our top priorities:
    - **Encryption**: All data encrypted with AES-256
    - **Compliance**: GDPR, FERPA, and SOC 2 Type II compliant
    - **Data Retention**: Configurable retention policies
    - **Access Control**: Strict role-based access controls
    - **Audit Logs**: Comprehensive activity tracking
  </Accordion>

  <Accordion title="Can candidates take assessments on mobile devices?">
    While candidates can access basic assessments on mobile devices, we recommend desktop/laptop computers for the best experience. Advanced proctoring features require desktop browsers with camera and microphone access.
  </Accordion>
</AccordionGroup>

## Candidate Experience

<AccordionGroup>
  <Accordion title="How do candidates access assessments?">
    Candidates receive assessment invitations via email containing:
    - Unique assessment link
    - Login credentials (if required)
    - Instructions and system requirements
    - Technical support contact information
  </Accordion>

  <Accordion title="What if a candidate has technical issues?">
    We provide multiple support options:
    - **System Check**: Pre-assessment technical requirements check
    - **Practice Mode**: Demo assessments to test setup
    - **Live Support**: Chat support during assessments
    - **Troubleshooting Guides**: Common issue resolution
    - **Alternative Methods**: Backup assessment options
  </Accordion>

  <Accordion title="Can assessments be paused and resumed?">
    This depends on assessment settings:
    - **Pause Allowed**: Candidates can pause and resume within time limits
    - **No Pause**: Assessment must be completed in one session
    - **Proctor Control**: Proctors can pause for technical issues
    - **Auto-save**: Progress is automatically saved
  </Accordion>

  <Accordion title="How long are assessment results available?">
    Result availability depends on organization settings:
    - **Immediate**: Results shown immediately after completion
    - **Delayed**: Results released after review period
    - **Scheduled**: Results released on specific date/time
    - **No Results**: Scores not shared with candidates
  </Accordion>
</AccordionGroup>

## Analytics & Reporting

<AccordionGroup>
  <Accordion title="What analytics are available?">
    Skillment provides comprehensive analytics:
    - **Assessment Analytics**: Completion rates, score distribution, time analysis
    - **Question Analytics**: Difficulty analysis, discrimination index, response patterns
    - **Candidate Analytics**: Individual performance, progress tracking, behavioral insights
    - **Organization Analytics**: Cross-assessment trends, team performance metrics
  </Accordion>

  <Accordion title="Can I export data and reports?">
    Yes, multiple export options are available:
    - **PDF Reports**: Formatted assessment and candidate reports
    - **Excel/CSV**: Raw data for further analysis
    - **API Access**: Programmatic data retrieval
    - **Custom Reports**: Scheduled automated reports
  </Accordion>

  <Accordion title="How is assessment performance calculated?">
    Performance metrics include:
    - **Raw Score**: Total points earned
    - **Percentage Score**: Score as percentage of total marks
    - **Percentile Rank**: Performance relative to other candidates
    - **Time Efficiency**: Score per unit time spent
    - **Question-level Analysis**: Performance by topic/skill area
  </Accordion>

  <Accordion title="Can I compare candidate performance across assessments?">
    Yes, cross-assessment analytics include:
    - **Candidate Profiles**: Performance history across multiple assessments
    - **Skill Mapping**: Performance by competency areas
    - **Trend Analysis**: Performance improvement over time
    - **Benchmarking**: Comparison against organization standards
  </Accordion>
</AccordionGroup>

## API & Integrations

<AccordionGroup>
  <Accordion title="Does Skillment have an API?">
    Yes! Skillment offers a comprehensive REST API that allows you to:
    - Create and manage assessments programmatically
    - Invite candidates and track progress
    - Retrieve results and analytics data
    - Configure proctoring settings
    - Manage organization and team members
  </Accordion>

  <Accordion title="What integrations are available?">
    Skillment integrates with popular platforms:
    - **LMS Integration**: Canvas, Moodle, Blackboard
    - **HR Systems**: BambooHR, Workday, ADP
    - **Communication**: Slack, Microsoft Teams, Discord
    - **Analytics**: Google Analytics, Mixpanel
    - **Webhooks**: Real-time notifications to your systems
  </Accordion>

  <Accordion title="Is there an SDK available?">
    Yes, we provide official SDKs for:
    - **JavaScript/Node.js**: Full-featured SDK with TypeScript support
    - **Python**: Comprehensive Python library
    - **PHP**: Laravel and standalone PHP support
    - **Java**: Spring Boot integration
    - **REST API**: Direct HTTP API access for any language
  </Accordion>

  <Accordion title="Can I white-label Skillment?">
    Enterprise customers can access white-label options:
    - **Custom Branding**: Complete UI customization
    - **Custom Domain**: assessments.yourcompany.com
    - **API Integration**: Embed into existing platforms
    - **Custom Features**: Tailored functionality development
  </Accordion>
</AccordionGroup>
